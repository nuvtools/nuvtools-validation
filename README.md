# Nuv Tools Validation Library

Nuv Tools Validation is a set of libraries for .NET 8 and .NET 9, designed to provide robust validation features for Web, Desktop, and Mobile (MAUI) applications.

## Libraries Overview

- **NuvTools.Validation**
  - Core validation library.
  - Provides static methods for validating Brazilian documents (CPF, CNPJ).
  - Includes regex patterns for email, Base64, and Data URI validation.
  - Offers formatting utilities for CPF and CNPJ.
  - Contains data annotation attributes for model validation.

- **NuvTools.Validation.AspNetCore.Blazor**
  - Extension library for Blazor applications.
  - Integrates with FluentValidation and ASP.NET Core forms.
  - Enables advanced validation scenarios in Blazor projects.

## Supported Frameworks

- .NET 8
- .NET 9
